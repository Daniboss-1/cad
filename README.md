# Aether CAD (Phase 0)

Aether CAD is a next-generation parametric CAD application built with Next.js, Three.js, and Manifold-3D.

## Features (Phase 0)
- **Parametric History Tree**: Manage your design as a list of editable nodes.
- **Command-K Nexus**: A spotlight-style search shell for quickly adding primitives.
- **Primitive Support**: Support for Box, Sphere, Cylinder, and Torus.
- **Real-time CSG**: Immediate visual feedback for boolean operations.
- **Responsive Sidebar**: Intuitive controls for primitive parameters.
